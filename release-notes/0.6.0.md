# SWUST Code v0.6.0

龙山灵码 0.6.0 是当前主线基于 MiMo-Code 能力完成重建后的首个正式发布版本。
这次发布的方向很明确：以 MiMo-Code 为底座，保留 SWUST-Code 的品牌、交互风格和已有增强能力，而不是回退到旧 OpenCode 主线。

## Highlights

- 完成 SWUST-Code 到 MiMo-Code 基座的主线迁移，当前主线能力以 MiMo 体系为准。
- 保留并继续强化龙山灵码品牌，包括命令名、配置目录、环境变量和 TUI 主题风格。
- 新增子智能体个性化配置能力，可按项目为现有子智能体覆盖模型、思考强度和最大执行步数。
- 补齐主线文档、README 与文档站信息，使当前仓库、npm 包和发布页面保持一致。
- 建立 npm Trusted Publishing 与 GitHub Release 自动发布链路，为后续版本发布提供稳定基础。

## What is included in 0.6.0

- MiMo-Code 主线能力继承
  - 持久化记忆
  - checkpoint / context reconstruction
  - actor / subagent 编排
  - goal / compose / dream / distill 等内建工作流
- SWUST-Code 主线增强延续
  - 龙山灵码品牌替换与主题整合
  - 侧边栏与命令区的中文化与交互打磨
  - 来自旧主线的关键增量功能迁移与兼容修复
- 发布工程化
  - GitHub Release 自动创建与资产上传
  - npm 包自动发布
  - 多平台二进制包发布到 `@swust-code/*`

## Install

```bash
npm install -g @swust-code/cli
swust-code
```

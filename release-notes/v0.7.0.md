# SWUST Code v0.7.0

龙山灵码 0.7.0 延续 MiMo-Code 主线，继续强化 SWUST 层的终端交互、Provider 连接与工程化体验。

## Highlights

- 自定义 Provider 新增三种 API 端口连接方式：旧 OpenAI、 新 OpenAI、Anthropic。
- 自定义供应商向导补齐连接类型选择，并保持原有配置写入流程。
- 新增对应的测试覆盖，并修复新 OpenAI Responses 路径的类型收窄。

## Install

```bash
npm install -g @swust-code/cli
swust-code
```
